## 应用介绍

AI记账是一款专为手机设计的智能记账应用，支持通过自然语言对话完成记账操作，让记账变得简单快捷。

### 主要功能
- **AI智能记账**：用日常语言描述收支，AI自动解析
- **多种主题**：支持浅色科幻清新、2026马年等多种精美主题
- **多账户管理**：现金、银行卡、支付宝、微信等
- **详细统计**：趋势图、分类图、月度对比
- **数据安全**：本地加密存储，支持PIN码和生物识别

## 快速开始

### 1. 首次设置
1. 打开应用，设置4位PIN码
2. 设置用户名（可选）
3. 选择喜欢的主题
4. 配置AI模型（如需使用AI功能）

### 2. 快速记账
在主界面点击右下角AI按钮，对着AI助手说：
```
"早饭花了15元"
"工资收入8000元"
"给妈妈转了500元"
```

AI会自动解析并确认记账。

## AI智能记账

### 基本对话记账
在AI助手页面输入自然语言描述：

**收入记账**：
```
"今天工资收入5000元"
"收到红包200元"
```

**支出记账**：
```
"午饭花了35元"
"买衣服用了500元"
"超市购物86.5元"
```

**转账记账**：
```
"从工商卡转了1000到储蓄卡"
```

### 连续对话
AI助手支持连续对话：
```
你：上周花了多少钱？
AI：上周支出共1234.5元
你：具体花在哪些地方？
AI：餐饮568元，交通200元，购物466.5元
```

### 模糊查询
```
"查一下上个月的花销"
"看看这个月收入多少"
"今年总共存了多少钱"
```

## 手动记账

### 添加交易
1. 点击右下角"+"按钮
2. 选择交易类型（收入/支出/转账）
3. 填写金额、分类、账户、时间
4. 点击保存

### 编辑交易
1. 在明细页面点击任意交易
2. 修改相关信息
3. 保存或删除

### 转账操作
1. 点击"+"按钮
2. 选择"转账"
3. 选择源账户和目标账户
4. 输入金额
5. 确认转账

## 账户管理

### 查看账户
在"设置"页面点击"账户管理"：
- 查看所有账户余额
- 查看账户明细

### 添加账户
1. 进入账户管理
2. 点击"添加账户"
3. 选择账户类型（现金/银行卡/支付宝等）
4. 输入账户名称和初始余额

### 账户类型
- 💵 现金
- 🏦 银行卡
- 📱 支付宝
- 💬 微信
- 💳 信用卡
- 其他自定义类型

## AI对话助手

### 对话界面
- 底部输入框发送消息
- 支持语音输入（点击麦克风图标）
- 支持图片发送（点击图片图标）

### AI功能
- **记账**：自然语言记账
- **查询**：查询收支、余额
- **建议**：获取理财建议
- **闲聊**：日常对话

### 使用示例
```
记账："中午请客花了200元"
查询："这个月在餐饮上花了多少钱？"
建议："我应该如何节省开支？"
```

## 数据安全

### PIN码保护
- 首次使用设置4位PIN码
- 每次打开应用需要输入PIN
- 可在设置中修改或关闭

### 生物识别
- 支持指纹识别
- 支持面部识别
- 可在设置中开启/关闭

### 数据加密
- 数据库使用SQLCipher加密
- 所有数据存储在本地
- 不上传至服务器

### 数据备份
1. 进入"设置"
2. 点击"导出数据"
3. 选择导出格式（Excel/CSV）
4. 保存到本地

## 常见问题

### Q: AI助手不回答问题？
A: 请检查AI设置中的API Key是否配置正确。

### Q: 记账金额显示不正确？
A: 检查默认货币设置，确保与实际一致。

### Q: 如何删除交易？
A: 在明细页面点击交易，进入详情后点击删除按钮。

### Q: 账户余额不对？
A: 检查是否有遗漏的转账或调整操作。

### Q: 主题不生效？
A: 部分主题可能需要重启应用后生效。

### Q: 如何联系开发者？
A: 在设置页面有反馈入口，欢迎提出建议。
